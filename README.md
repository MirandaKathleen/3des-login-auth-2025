# Login - auth

## Atividade em grupo de até 3 integrantes.

- Testar a api com **insomnia**.
- Estudar e documentar a estrutura do projeto.

- Documentar descrição do funcionamento utilizando **UML DA(Diagrama de Atividades)**.

- VsCode
- Insomnia
